CCOM Documentation

Notes

CCOMs (custom commands) are written in PHP syntax and are interpreted by a limited PHP interpreter. Most commands are disabled for security reasons and because they're irrelevant for our use case.

Command usage

.ccom add hello print('world');
This will add the command "hello" which executes the code "print('world');"

.ccom remove hello
This will remove the command "hello." You can only remove commands that you've added.

Using variables

There are a few predefined variables you can use:

  • $input - The input that was used to execute the command
  • $from - The user that executed the command

.ccom add example $var = "hello world"; print($var);
The code above assigns "hello world" to the variable $var and prints it out

variables will only last as long as the command is being executed. After execution all variables are destoryed

Functions

There are a limited number of functions you can use:

  • argument( index ) - Gets part of the input as an argument
  • wwwget( url ) - Gets the source of a url as a string

You may also use the following internal PHP functions

  • time
  • date
  • sleep
  • print
  • rand
  • str_replace
  • str_pos
  • explode
  • date
  • substr
